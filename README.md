# Next.js Gemini Project

A **Next.js** project with interactive React components and Gemini API integration for AI-powered responses.

---

## Features

- Built with **Next.js 16+** and **React 19+**  
- Animated text using the `BlurText` component  
- Interactive UI lines with `MagnetLines` component  
- Input form connected to **Gemini API**  
- Responsive and modern UI  

---

## Prerequisites

- Node.js `>= 20`  
- npm or yarn  
- Git  

---

## Getting Started

1. **Clone the repository**

```bash
git clone <your-repo-url>
cd <your-project-folder>
```

Install dependencies

```
npm install
# or
yarn install

```
```
Setup environment variables

Create a .env.local file in the project root and add your Gemini API key:

NEXT_PUBLIC_GEMINI_API_KEY=your_gemini_api_key_here
```


Running the Project
```
npm run dev
# or
yarn dev
```
